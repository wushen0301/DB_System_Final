# 三下資料庫系統設計期末專案

透過 Python ，使用 NiceGUI 及sqlite3建立資料庫應用系統。

## 專案檔案說明

- 請將**所有檔案放置於同一個資料夾中**
- 主要執行檔：
  - `demo_data.py`：用於插入示範（demo）資料
  - `app.py`：主程式，啟動後可透過瀏覽器操作系統

## 開啟請使用終端開啟

## 使用前準備

在執行本專案前，請先確認已安裝所需套件：

```bash
pip install nicegui
```
